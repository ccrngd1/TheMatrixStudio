# TheMatrix Simulation Studio

A standalone, distributable multi-agent simulation tool — spiritual successor to Stanford's
`generative_agents`, built on a hand-rolled async LiteLLM engine. Provider-agnostic (BYO key),
runs as a local Python app or a Docker container.

**Status:** Phase 0 (extract & de-couple) — in build.
